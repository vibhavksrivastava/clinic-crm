# Walk-in Prescription Generation - Quick Start Guide

## 🎯 How to Generate Prescriptions from Walk-ins

### Step 1: Create a Walk-in
1. Go to **Walk-ins** page
2. Click **"New Walk-in"** form on the left
3. Enter patient details:
   - Name
   - Phone number
   - Address
   - Select doctor (optional)

### Step 2: Start the Walk-in
1. Click **"Start"** button on the walk-in card
2. Status changes to "In Progress"

### Step 3: Record Patient Details
1. **Add Tests**: Click "🧪 Additional Tests" to add medical tests
2. **Add Vitals**: Click "🩺 Vitals" to record vital signs
3. **Add Medicines**: Click "💊 Medicines" to add medications
   - Enter medicine name
   - Enter dosage (e.g., 500mg)
   - Enter frequency (e.g., 3x daily)

### Step 4: Complete the Walk-in
1. Click **"✓ Complete"** button
2. If medicines were added:
   - A prescription is **automatically generated**
   - Success message appears: "Walk-in completed and prescription generated successfully!"
3. Walk-in status changes to "Completed"

### Step 5: View the Prescription
After completion, you can view the prescription in multiple ways:

#### Option A: From Walk-in Card
1. Look for completed walk-in
2. Click **"💊 Prescription"** button (appears if medicines were added)
3. You'll be directed to the Prescriptions page

#### Option B: From Prescriptions Page
1. Go to **Prescriptions** page
2. Look for prescriptions with **"🚶 Walk-in Prescription"** label
3. Click to view prescription details
4. Click **"Print"** or **"Download"** to get PDF

#### Option C: From Patient Page
1. Go to **Patients** page
2. Click on patient name
3. Scroll to **"💊 Prescriptions"** section
4. All prescriptions (from appointments and walk-ins) are listed
5. Click **"View Prescription"** button

## 📋 Prescription Details

When you view a prescription generated from a walk-in:
- Shows **"🚶 Linked to Walk-in"** badge
- Lists all medicines prescribed
- Shows patient vitals (if recorded)
- Shows doctor's name
- Shows issue date
- Can be printed or shared

## ✅ Key Features

✨ **Automatic Generation**
- Prescriptions are created automatically when completing walk-in with medicines
- No need to manually create prescription afterward

🔗 **Linked Data**
- Prescription is linked to walk-in
- Can trace back from prescription to original walk-in
- All patient data is kept together

🏥 **Multi-View Access**
- View from walk-in card
- View from prescriptions page
- View in patient history
- All show the walk-in connection

📄 **Professional Format**
- Prescription is formatted for printing
- Includes clinic information
- Shows all required medical details
- Ready to give to patient

## ⚠️ Important Notes

- **Medicines Required**: Prescriptions are only generated if at least one medicine is added
- **Doctor Required**: Walk-in should have a doctor assigned for proper tracking
- **Vitals Optional**: You can record vitals but they're not required for prescription generation
- **Tests Optional**: Additional tests don't affect prescription generation

## 🔄 Workflow Comparison

### Before (Manual Process)
1. Complete walk-in with medicines
2. Navigate to prescriptions page
3. Click "Create Prescription"
4. Manually enter all patient data again
5. Manually enter medicines again
6. Submit prescription

### After (Automatic Process)
1. Complete walk-in with medicines
2. ✅ Prescription automatically generated!
3. View prescription in prescriptions page or walk-in card
4. Print and give to patient
